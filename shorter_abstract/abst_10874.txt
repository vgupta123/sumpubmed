documentation for the birch and locally-added programs is merged in a hierarchical set of web pages.
these include tools for integrating locally-installed programs and databases into birch, and for customizing the local birch system to meet the needs of the user base.
birch provides a versatile organizational framework for managing software and databases, and making these accessible to a user base.
birch  is an organizational framework for delivering bioinformatics resources to a user group, scaling from a single lab to a large institution.
the birch core distribution includes many popular bioinformatics programs, unified within the gde  graphic interface.
at the same time, system administrators tasked with maintaining these tools do not always appreciate the needs of research biologists.
of equal importance, birch provides the system administrator with tools that simplify the job of managing a multiuser bioinformatics system across different platforms and operating systems.
