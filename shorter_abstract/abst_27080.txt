maffilter is an open source software available at http://bioweb.me/maffilter.
maffilter is a command-line driven program written in c++ that enables the processing of genome alignments stored in the multiple alignment format in an efficient and extensible manner.
maffilter is a highly efficient and flexible tool to analyse multiple genome alignments.
genome sequence alignments are complex structures containing information such as coordinates, quality scores and synteny structure, which are stored in multiple alignment format  files.
